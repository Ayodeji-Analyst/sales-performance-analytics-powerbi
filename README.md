An interactive Power BI dashboard designed to analyze sales performance, profitability, customer purchasing behavior, and product performance through dynamic visualizations and business intelligence reporting.

📌 Project Overview

This project showcases an interactive Sales Performance Analytics Dashboard built using Microsoft Power BI. The dashboard transforms raw sales data into meaningful insights by tracking key performance indicators (KPIs), identifying sales trends, and highlighting customer and product performance.

The goal of this project is to help business stakeholders monitor sales performance and make informed, data-driven decisions.

🎯 Business Problem

Businesses generate large volumes of sales data daily, making it difficult to identify performance trends without effective reporting tools.

This dashboard was developed to answer key business questions such as:

Which products generate the highest sales?
Which customers contribute the most revenue?
How much profit is generated over time?
Which product categories perform best?
What are the daily sales trends?
🎯 Project Objectives
📈 Monitor Total Sales, Profit, and Quantity Sold.
📊 Analyze sales by product category and sub-category.
👥 Identify top-performing customers.
📅 Track sales trends over time.
🎛️ Build an interactive dashboard with dynamic filtering.
💡 Provide actionable business insights through visualization.
